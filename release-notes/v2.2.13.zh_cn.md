# Farming v2.2.13

[English](./v2.2.13.md)

这个版本显著改善 Farming CRT 结构化 Chat 的可读性，同时保留键盘优先的控制室气质；并修复 Main Agent 恢复与首次聚焦边界、加固异步 Terminal Link，使用新的 CRT Billing 控制室更新 README 主宣传图。

## Highlights

- **CRT Chat 安全 Markdown**：Agent Reply 现在支持 GitHub Flavored Markdown、代码语法高亮、KaTeX 公式和按需加载的 Mermaid 图。Raw HTML 继续禁用，危险链接会被移除；非法图表保留源码并进入有界错误状态。
- **更易读的 CRT 字体系统**：结构化 Chat、操作控件、元信息和 Agent Preview 使用更紧凑、统一的字号。代码保留有价值的语法颜色，Preview 文字则移除影响清晰度的磷光扩散。
- **可靠进入 Main Agent**：启动 Main Agent 时，如果权威 ID 临时丢失，会重新认领已经运行的 Main 进程。从其他 Session 打开 Main Agent 后，会在首次权威屏幕快照稳定时保持 Terminal Input Focus，同时不会抢走用户已经主动选择的控件。
- **更安全的 Terminal Link 与更确定的浏览器覆盖**：延迟完成的 Terminal Link 只允许作用于创建它的 Session；Files、Navigation、Display Flow 和 Focus Recovery 使用更稳定的测试契约。
- **CRT 控制室宣传图**：README 首张产品图更新为当前 CRT Billing 控制室。

## Verification

- Backend 与源码契约测试套件
- TypeScript Typecheck 与 ESLint
- CRT Markdown、KaTeX、Mermaid 和 Main Agent Focus Recovery 的定向 Chromium 覆盖
- `/farming` Production Build

## Upgrade

```bash
npm install --global farming-code@2.2.13 && farming daemon
```
