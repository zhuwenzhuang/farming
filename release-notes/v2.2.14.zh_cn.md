# Farming v2.2.14

[English](./v2.2.14.md)

这是一个小型跟进版本：修复键盘焦点竞态，并围绕默认的 Farming Code 体验更新 Farming 的公开介绍。

## 修复

- **稳定的 New Agent 焦点**：关闭 New Agent 弹窗后立即重新打开时，旧的延迟焦点恢复回调不再把键盘焦点移到新弹窗背后。

## 文档

- README 重新以 Farming Code 这个默认远程 Web 工作台为主线，提供直接的启动路径，并更新桌面、Files、移动 Chat、Agent 活动和可选 CRT 截图。
- 仓库简介直接说明远程 Web 工作台定位，同时保留 CRT 作为可选界面。

## 维护

- 在不改变 Chromium 浏览器测试集合的前提下，将其拆分到六个相互隔离的 CI shard。
- 移除 Terminal Renderer 测试对文档固定措辞的依赖，并让焦点、Transcript 滚动和布局检查更确定，同时保留原有产品行为断言。

## 验证

- TypeScript Typecheck、ESLint，以及 184 个 Backend/源码契约测试
- 六个隔离 shard 上共 172 个 Chromium case；发布基线无 retry、无 flaky
- `/farming` Production Build 与 Release Workflow 包产物 Smoke Test

## 升级

```bash
npm install --global farming-code@2.2.14 && farming daemon
```
