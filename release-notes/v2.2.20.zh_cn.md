# Farming v2.2.20

[English](./v2.2.20.md)

本次补丁发布改进 CRT Token Telemetry，并让 Token Usage 交互更稳定。

## 修复

- CRT 顶栏 `TOK/MIN` 改用 Provider 上报的 Token Telemetry，不再根据 Terminal 输出量估算。
- CRT Billing Live 改为一小时窗口、一分钟桶、15 秒有界刷新，并显示最近采样时间。
- DAYS 与 LIVE 分离选中态和键盘焦点态，只有当前 Tab 显示为选中。
- Usage activity 日历增加可取消的 200ms 悬停延迟。快速扫过日期时不再触发中间请求或 Loading 闪烁；点击、键盘和手机日期选择仍然立即生效。
- CRT 日历方向键与可见周网格一致，并避免 Main Agent 引导打断恢复后的非 Agent 页面。

## 内部调整

- 共享 Transcript 实现由 Codex-first 命名改为 Agent-first，同时继续保留明确的 Provider 专属适配层。
- CRT Settings 移除已经失效的固定 Session Engine 说明。

## 验证

- 187 个 Backend 与源码契约测试
- TypeScript Typecheck 与 ESLint
- 3 个 Chromium 定向流程：Usage 悬停、CRT Provider Telemetry、Billing Live

## 升级

```bash
npm install --global farming-code@2.2.20 && farming daemon
```
